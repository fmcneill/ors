                        README
=====================================================

This is an extension to ORS (Ontology Repair System),
which enables human interaction in ORS.

Users can approve or disapprove the repair requests,
and provide additional information to instantiate new
arguments.

How to start the Prolog program:

1. ./agent_environment/scenarios/shopping/run.sh

2. startpa.

How to start the Java GUI:

1. cd GUI/

2. ./run.sh
